## Project Overview

This project analyzes a global survey of data professionals to understand career trends, salaries, tools, and job satisfaction. The data was cleaned using Power BI Power Query and visualized through an interactive dashboard.



## Dataset

Survey responses include:

- Job Title

- Salary

- Programming Language

- Country

- Work-Life Balance

- Salary Satisfaction

- Difficulty Breaking into Data

- Gender

- Age



## Data Cleaning

- Removed empty and unnecessary columns

- Standardized categorical values

- Cleaned survey metadata

- Ensured data consistency using Power Query



## Business Questions Answered

1. Average salary by job title

2. Most commonly used programming languages

3. Geographic distribution of respondents

4. Work-life balance satisfaction

5. Salary satisfaction

6. Difficulty entering the data field

7. Gender distribution



## Tools Used

- Power BI

- Power Query

- Data Visualization



## Key Insights

- Salary varies significantly by role

- Breaking into data is challenging for many professionals

- Work-life balance and salary satisfaction vary across roles
